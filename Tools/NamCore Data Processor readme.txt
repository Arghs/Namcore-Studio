NamCore Data Processor will extract necessary data from dbc/db2 files and creates
csv files that need to be embedded into the libnc project).

Requires following dbc/db2 files:

/DE/ folder:
#German localization dbc files

- Achievement.dbc
- Achievement_Category.dbc
- Faction.dbc
- Item-sparse.db2
- SkillLine.dbc
- Spell.dbc
- SpellItemEnchantment.dbc


/EN/ folder:
#English localization dbc files

- Achievement.dbc
- Achievement_Category.dbc
- Faction.dbc
- Item-sparse.db2
- SkillLine.dbc
- Spell.dbc
- SpellItemEnchantment.dbc

/Shared/ folder:

- GlyphProperties0.dbc (GlyphProperties.dbc WotLK)
- GlyphProperties1.dbc (GlyphProperties.dbc Cataclysm)
- GlyphProperties2.dbc (GlyphProperties.dbc MoP)
- Item.db2
- ItemDisplayInfo.dbc
- SpellEffect.dbc
- SpellIcon.dbc
